# CROWD Oracle

**The Social Intelligence Layer for DeFi.**

CROWD is a decentralized social oracle where participants predict the performance of new token launches. By aggregating crowd wisdom, CROWD generates a "True Market Consensus" on asset potential.

## Features

-   **Oracle Terminal:** Real-time interface for tracking and predicting asset performance.
-   **Antigravity Mode:** Immersive, animated UI experience.
-   **Solana Integration:** Connect your wallet to stake $CROWD and lock in predictions.
-   **Neo-Brutalist Design:** High-contrast, futuristic visual aesthetic.

## Tech Stack

-   **Frontend:** Next.js, React, Tailwind CSS
-   **Blockchain:** Solana Web3.js, Wallet Adapter
-   **Styling:** Custom Neo-Brutalist & Cyberpunk themes

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Run the development server: `npm run dev`
